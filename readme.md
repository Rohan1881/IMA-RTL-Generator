# IMA - Intelligent Memory Architecture

An automated RTL generation and verification framework.

## Features
- Smart input parsing
- Jinja2-based RTL generation
- Multi-memory support (SRAM/DRAM)
- Automated testbench generation
- Simulation with waveform (VCD)
- Power, CDC, and Lint reports
- Summary generation

## Flow
Input → RTL → Testbench → Simulation → Reports
